# Motivatiebrief — Junior Data Engineer bij DAS

Beste wervingsteam van DAS,

Met veel interesse reageer ik op de vacature voor Junior Data Engineer bij DAS in Amsterdam.

Als Software Engineer bij Actemium (onderdeel van VINCI Energies) werk ik dagelijks met Python en SQL voor het bouwen en onderhouden van data-gestuurde applicaties en backend services. De missie van DAS — het recht toegankelijk maken voor iedereen — en de rol van data daarin spreken mij zeer aan.

Mijn relevante ervaring:
- **Python**: Dagelijkse ontwikkeling met Python/Flask, REST API's, data processing en automatisering
- **SQL**: SQL Server voor data-opslag, querying en rapportage bij Actemium
- **Azure**: Ervaring met Azure cloud platform, opgedaan tijdens mijn stage bij ASML (Azure Kubernetes Service, Azure DevOps)
- **ETL & Data pipelines**: Ervaring met het verwerken en transformeren van data uit verschillende bronnen
- **Tooling**: Git, CI/CD, Docker, Pytest voor geautomatiseerd testen

Tijdens mijn stage bij ASML heb ik performance testing frameworks gebouwd met Python, Pytest en Locust, gedeployed op Azure Kubernetes Service. Dit heeft mij vertrouwd gemaakt met cloud-infrastructuur en het werken met grote datasets. Bij mijn AI-project CogitatAI heb ik ervaring opgedaan met data-gestuurde applicaties en machine learning concepten.

Ik heb een BSc in Software Engineering behaald aan Fontys Hogescholen en beheers zowel Nederlands als Engels. Ik ben leergierig en enthousiast om mijn data engineering vaardigheden verder te ontwikkelen binnen het team van DAS.
